# Contributing Guidelines

Thank you for your interest in contributing to the Simple Calculator project!

## How to Contribute

1. **Fork the repository**
2. **Create a new branch** for your feature or fix
3. **Make your changes**
4. **Test your changes** thoroughly
5. **Commit with clear messages**
6. **Push to your branch**
7. **Create a Pull Request**

## Areas for Contribution

### Features to Add
- [ ] Scientific calculator functions (sin, cos, tan, sqrt, etc.)
- [ ] Calculator history/memory functions
- [ ] Theme switcher (dark mode, light mode)
- [ ] Sound effects on button clicks
- [ ] Undo/Redo functionality
- [ ] Number formatting (comma separators)

### Improvements
- [ ] Better error messages
- [ ] Accessibility improvements
- [ ] Performance optimizations
- [ ] Additional keyboard shortcuts
- [ ] Touch gesture support for mobile

### Documentation
- [ ] Better code comments
- [ ] API documentation
- [ ] Video tutorials
- [ ] More examples

## Code Style

- Use clear, meaningful variable names
- Add comments for complex logic
- Follow consistent indentation (2 spaces)
- Keep functions small and focused
- Test new features before submitting

## Reporting Issues

If you find a bug, please:
1. Search existing issues first
2. Provide a clear description
3. Include steps to reproduce
4. Mention your browser and OS

## Questions?

Feel free to open an issue for questions or discussion!
