# Simple Calculator

A beautiful, fully functional calculator built with HTML, CSS, and JavaScript.

## Features

- ✨ Modern, responsive design with gradient background
- 🎯 Basic arithmetic operations: Addition, Subtraction, Multiplication, Division, Modulo
- ⌨️ Full keyboard support
- 🔧 Delete last digit functionality
- 🚫 Division by zero error handling
- 📱 Mobile-friendly layout
- 🎨 Smooth animations and hover effects

## How to Use

1. Open `index.html` in your web browser
2. Click buttons or use your keyboard to enter numbers and operations
3. Click "=" or press Enter to calculate the result
4. Click "C" or press Escape to clear the calculator
5. Click "DEL" or press Backspace to delete the last digit

## Keyboard Shortcuts

- **Numbers**: 0-9
- **Operators**: +, -, *, /, %
- **Calculate**: Enter or =
- **Clear**: Escape
- **Delete Last**: Backspace
- **Decimal Point**: .

## Files

- `index.html` - HTML structure
- `style.css` - Styling and layout
- `script.js` - Calculator logic and functionality

## Live Preview

Open `index.html` in your browser to use the calculator immediately. No installation required!

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
